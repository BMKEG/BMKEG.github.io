The Biomedical Knowledge Engineering Group (BMKEG)

Open source code developed by BMKEG@ISI

We are an open source, biomedical informatics software development shop, funded from academic sources with a strong commitment to develop practical knowledge engineering systems to assist and support biomedical scientists in their everyday work.

Here we describe the software currently being developed and actively maintained in our GitHub repositories for end users. So, you will find user manuals, walkthroughs, simplified installation instructions and everything we think you might need to install and run our code easily out of the box. Our main GitHub page has lots of information for developers, so if you want to dive into the code itself, you should read this guide to our codebase.

Projects

The SciKnowMine Triage System

A web application that permit biocurators to manage collections of scientific articles and to accelerate the process of `docutment triage' (sorting them into categories of interest for subsequent detailed curation).

[Project Page] [Source Code] [Installers]

The Layout Aware PDF Text Extraction System

A low-level java library for accurately extracting text from PDF files and additional web-based tools to improve performance of the text extraction.

[Google Code (deprecated)] [Source Code]

Knowledge Engineering from Experimental Design

A general-purpose, ontology-aware knowledge engineering methodology for biomedical data that permits the construction of biomedical databases derived only from a description of the underlying experimental protocol used to gather the data.

[Project Page] [Source Code]

The View Primitive Data Model framework

A system scaffolding set of java-libraries that permit the agile development of web-applications. This is the foundation for all of our software tools and permits us to develop specialized database applications for scientists rapidly and efficiently.
